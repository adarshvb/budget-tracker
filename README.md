# Budget Tracker

A simple personal budget tracker built as a single HTML file with inline CSS and JavaScript.

## Features (v1)

- Add an expense with description, amount, and category
- See a list of all expenses
- See the running total spent
- Delete an expense
- Data persists in the browser via `localStorage`

## How to run

Open `budget-tracker.html` in any modern web browser. That's it — no install, no build step.

## Tech

- HTML + inline CSS + vanilla JavaScript
- `localStorage` for persistence (no backend)
